# Privacy Policy for TL;DR this PR

TL;DR this PR is a Chrome extension that helps users analyze GitHub pull requests by generating summaries and risk insights.

## Data Collection

This extension does not collect, store, or share any personal user data.

## What Data is Processed

When the user clicks the extension button on a GitHub pull request page, the extension reads the following content from the page:

- Pull request title and description
- Commit messages
- File changes and diff content

This data is used only to generate summaries and analysis.

## Third-Party Services

To generate summaries, the extension sends the pull request content to an external API (OpenRouter).

- No personal user data is sent
- The data is not stored by the extension
- The API key is stored locally in the user’s browser

## Data Storage

- API keys are stored locally using Chrome storage
- Generated summaries may be cached locally to reduce repeated API calls
- No data is stored on external servers

## Data Sharing

The extension does not sell, transfer, or share user data with third parties, except as required to process the analysis request via the OpenRouter API.

## User Control

Users can:

- Remove the extension at any time
- Clear stored data via browser settings

## Changes

This privacy policy may be updated in the future. Continued use of the extension implies acceptance of any updates.
